Authenticating a RHEL client to the network by using the 802.1X standard with a certificate stored on the file system

Port-based Network Access Control (NAC), based on the IEEE 802.1X standard, protects a network from unauthorized clients. If the network uses Extensible Authentication Protocol Transport Layer Security (EAP-TLS), you require a certificate to authenticate the client.

  • Configuring 802.1X network authentication on an existing Ethernet connection by using nmcli
  • Configuring a static Ethernet connection with 802.1X network authentication by using nmstatectl
  • Configuring a static Ethernet connection with 802.1X network authentication by using the network RHEL system role
  • Configuring a wifi connection with 802.1X network authentication by using the network RHEL system role
